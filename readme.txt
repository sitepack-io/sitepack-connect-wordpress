Connect your eCommerce store with external APIs, using SitePack Connect. Import products with stock information and export orders to your favorite software.

== Description ==

Automate your eCommerce store with the SitePack Connect plugin. Create import sources for products and create them automatically in your WooCommerce shop.

You can also export orders and integrate with third party software.
